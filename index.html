Chat Widget

Chat estilo WhatsApp com inteligência RAG para capturar e qualificar leads na sua landing page de CBD.

✅ ANVISA Regulamentado ✅ CFM Autorizado 100% Online RAG Integrado

✨ Funcionalidades do Widget

🧠

RAG Integrado

Responde automaticamente sobre CBD, condições, legalidade, entrega, preços e muito mais com base no conhecimento da Luna Organics.

🎯

Qualificação de Lead

Fluxo conversacional que coleta nome, condição de saúde e WhatsApp antes de redirecionar para um especialista.

📲

Redirecionamento WhatsApp

Leads qualificados são direcionados ao WhatsApp da Luna Organics com mensagem pré-preenchida automaticamente.

⚡

Quick Replies

Botões de resposta rápida para condições de saúde, facilitando a interação e aumentando a conversão.

📊

Evento para CRM

Dispara evento JavaScript customizado luna:lead_qualified para integração com qualquer CRM ou analytics.

🎨

Design da Marca

Visual alinhado à identidade da Luna Organics com verde teal #349588, indicador online e animações suaves.

🚀 Como Instalar na Landing Page

💡 Demonstração ativa: O chat já está funcionando nesta página. Clique no botão verde no canto inferior direito para testar!
1

Abra o arquivo chat-widget.js

Edite a linha com WHATSAPP_NUMBER e coloque o número real da Luna Organics.

2

Configure o número do WhatsApp

var WHATSAPP_NUMBER = '5511999999999'; // Número real aqui
3

Adicione o script na landing page

Cole antes do </body> no HTML da sua landing:

<script src="chat-widget.js"></script>
4

Opcional: Integrar com CRM

document.addEventListener('luna:lead_qualified', function(e) {
  // e.detail = { name, whatsapp, condition, qualified }
  fetch('/api/leads', { method: 'POST', body: JSON.stringify(e.detail) });
});
